
# Ray-SOC LP Review Studio

## 🚀 サイトをWeb公開する最終ステップ

スクリーンショットの画面（Settings > Pages）で以下の操作をしてください。

### 1. 公開設定（リポジトリがPrivateの場合）
無料プランで公開するには、リポジトリを **Public（公開）** にする必要があります。
1.  左メニュー **[General]** をクリック。
2.  一番下の **[Danger Zone]** で **[Change visibility]** → **[Make public]** を実行。

### 2. ブランチの選択（重要！）
1.  左メニュー **[Pages]** に戻ります。
2.  **[Build and deployment]** > **[Branch]** の **「None」** ボタンをクリック。
3.  **「main」** を選択して **「Save」** をクリック。

これで数分以内に `https://ourakawa.github.io/LP-for-MA/` でアプリが公開されます。

## 🛡️ セキュリティについて
- リポジトリを **Public** にすると、ソースコードとLP案の内容が全世界に公開されます。
- 機密情報（本物のパスワードや未発表の極秘戦略など）が含まれていないことを確認してから公開してください。
- 公開したくない場合は、リポジトリはPrivateのまま **Vercel** 等の外部サービスでデプロイすることをお勧めします。

## 🛠 使い方
- **URL共有**: 公開されたURLに `#pattern-1` などを付けると特定の案を表示できます。
- **レビュー投稿**: 「GitHubレビュー」ボタンからIssueを作成してフィードバックを管理できます。
